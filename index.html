Web-Server monitoring & logging

View Project

This project presents a possible application which is able to test the availability of a URL. This is achieved by querying the user for a valid HTML URL which is then checked every 30 seceonds if the URL is still available. The result of the check will be written to a local log file. A user can stop the program by presseing ANY key available on the keyboard.

LinkedIn Company Follower Scraper

View Project

This script scrapes the follower list and their profile information of ANY company profile, if admin rights of the company are available, and saves the data into an excel file.
